# HaloOnlineIntel
Websocket for chat/stats

```
-rPass string
	Rcon Password
-rPort int
	Rcon Port (default 11776)
-sIP string
	Server Address (default "127.0.0.1")
-sPort int
	Server Port (default 11775)
-wsPort int
	Local Socket Port (default 5000)
```
